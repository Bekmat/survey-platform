Проект: Платформа онлайн-опросов (Survey Platform)

Краткое описание:
Простое Flask-приложение для создания опросов, прохождения опросов и экспорта результатов в Excel.

Зависимости:
- Flask
- pandas
- openpyxl

Установка и запуск (в PyCharm или в терминале):
1. Перейдите в папку проекта `survey_platform`.
2. Создайте виртуальное окружение (рекомендуется).
3. Установите зависимости:
   pip install -r requirements.txt
4. Запустите приложение:
   python app.py
5. Откройте браузер по адресу: http://127.0.0.1:5000

Файлы:
- app.py - основной код приложения
- templates/ - HTML-шаблоны (base.html, index.html, create.html, survey.html, results.html)
- requirements.txt - зависимости
